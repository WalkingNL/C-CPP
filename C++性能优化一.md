## 临时对象
在解释临时对象之前，我倒是想先谈谈临时对象的意义所在，它对性能产生怎样的影响呢？

### 临时对象对性能的影响
对象的创建与销毁对程序性能的影响很大。尤其那些处于复杂继承体系末端的类，其对象的构造及销毁更是如此。因此尽量减少创建不必要的对象。这似乎显而易见，但这里所说的对象更多意指临时对象。区别于显式定义的对象，如下代码。临时对象天生的不可见性，很容易造成一种错觉就是，它应该就被“忽略”，而且这不是我的错。
    
    class A
    {
    public:
        A() {}
        A(const A&) {} 
        
    int* a;
    };
