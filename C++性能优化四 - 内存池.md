## Memory pool
在C/C++中，**malloc**函数或者**new**运算符是用来进行内存分配的，可能我们都知道，它们非常的耗时。但到底有多耗时，对程序性能影响有多大，如果不亲眼见一次，真的很难相信。所以这篇文章，将介绍一个我在工作中遇到的案例，围绕这个案例，我们再来学习内存池。

这篇文章的计划是这样的：
  * 案例的简介
  * 内存池的种类
  * 固定内存池
  * 可变内存池
  * 多线程情况
